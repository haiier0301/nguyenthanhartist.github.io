## Image Guidelines

- **Format**: JPG for photographs/artworks
- **Quality**: High resolution for hero/featured images (1920px+), standard resolution for thumbnails (800-1200px)
- **Optimization**: Compress images for web without significant quality loss
- **Aspect Ratios**:
  - Hero: 16:9 or wider
  - Series banners: 3:2 or 16:9
  - Artworks: Varies based on actual artwork dimensions
  - News thumbnails: 4:3
  - Portrait: 3:4 or similar vertical format
