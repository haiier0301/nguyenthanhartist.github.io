Quick Start Guide

1

Prepare Your Images

Photograph all artworks in high quality. Images should be:

  • High resolution (2000px+ width for hero images)
  • Well-lit with accurate colors
  • Saved as JPG format
  • Compressed to 200-500KB per file
2

Organize Your Files

Rename your images according to the required filenames. See the complete list in:

images/IMAGE_GUIDE.txt

Examples: hero-artwork.jpg, peace-city-1.jpg, artist-photo.jpg

3

Upload Images

Place all renamed images into the images/ folder in this directory.

Important: Exact filenames are required. Check the IMAGE_GUIDE.txt file for the complete list of 28 required images.
4

View Your Website

Once images are added, open index.html in your web browser to view the complete gallery website.
