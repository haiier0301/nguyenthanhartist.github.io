# Nguyen Thanh - Art Gallery Website

A minimalist art gallery website inspired by whitecube.com, featuring clean design with generous white space, thin typography, and minimal animations.

## Design Features

- **Clean Gallery Aesthetic**: Off-white background (#fafaf8) with black text
- **Typography**: Inter/Helvetica Neue with font-weight 300-400
- **Minimal Design**: No icons, no borders, no underlines
- **Generous White Space**: Following professional gallery standards
- **Fully Responsive**: Mobile, tablet, and desktop optimized

## File Structure

```
├── index.html          # Home page
├── about.html          # About the artist
├── artworks.html       # Artworks gallery
├── exhibitions.html    # Exhibitions & awards
├── news.html          # News & press
├── contact.html       # Contact information
├── styles.css         # Main stylesheet
├── script.js          # Minimal JavaScript
├── images/            # Image folder (see below)
└── README.md          # This file
```

## Image Requirements

To complete the website, add the following images to the `images/` folder:

### Home Page (index.html)
- `hero-artwork.jpg` - Large hero image (2000px+ width recommended)

### About Page (about.html)
- `artist-photo.jpg` - Portrait of the artist (1200px+ width recommended)

### Artworks Page (artworks.html)

**Peace in the City Series:**
- `peace-city-hero.jpg` - Hero image for featured series
- `peace-city-1.jpg` through `peace-city-6.jpg` - Series artworks

**Silent Horizon Series:**
- `silent-horizon-hero.jpg` - Hero image for series
- `silent-horizon-1.jpg` through `silent-horizon-4.jpg` - Series artworks

**Urban Light Series:**
- `urban-light-hero.jpg` - Hero image for series
- `urban-light-1.jpg` through `urban-light-4.jpg` - Series artworks

### News Page (news.html)
- `news-spotlight-award.jpg` - Red Dot Miami award/booth photo
- `news-aaf-london-2023.jpg` - London 2023 booth photo
- `news-aaf-london-2022.jpg` - London 2022 booth photo
- `news-art-business.jpg` - Art Business News feature image
- `news-vnexpress.jpg` - VnExpress article image

## Image Specifications

- **Format**: JPG or PNG
- **Hero Images**: Minimum 2000px width, high quality
- **Artwork Images**: Minimum 1200px width, high quality
- **Portrait/News Images**: Minimum 800px width
- **Optimization**: Compress images for web (70-85% quality) for faster loading

## How to Use

1. **Add Your Images**: Place all artwork photos in the `images/` folder with the exact filenames listed above
2. **Open in Browser**: Open `index.html` in any modern web browser
3. **Customize Content**: Edit the HTML files to update text, titles, or add more artworks
4. **Styling**: Modify `styles.css` to adjust colors, spacing, or typography if needed

## Customization

### Colors
The main colors are defined in `styles.css`:
- Background: `#fafaf8` (off-white)
- Text: `#000` (black)
- Secondary text: `#333` or `#666` (gray)

### Typography
Default font stack: `'Inter', 'Helvetica Neue', 'Neue Haas Grotesk'`
- Headings: font-weight 400
- Body text: font-weight 300

### Spacing
White space is generous throughout. Main spacing values:
- Section margins: 80-160px
- Content padding: 40-80px
- Grid gaps: 40px
