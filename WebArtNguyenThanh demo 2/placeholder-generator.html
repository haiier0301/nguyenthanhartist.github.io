Nguyen Thanh Gallery - Image Setup Guide

Instructions for Adding Images

This website requires artwork photographs to be complete. Follow these steps:

  1. Take high-quality photos of all artworks (2000px+ width recommended)
  2. Save images as JPG format with the exact filenames listed below
  3. Place all images in the images/ folder
  4. Open index.html in your browser to view the complete website

Note: Until images are added, placeholder backgrounds will show where images should appear.

Quick Start

Once you have added all images to the images/ folder:

Image Specifications

  • Format: JPG (JPEG)
  • Quality: 85-90% compression
  • Color Space: sRGB
  • Hero Images: 2400 x 1600px minimum
  • Artwork Images: 1400 x 1000px minimum
  • News Images: 1000 x 700px minimum
  • File Size: Keep under 500KB per image
