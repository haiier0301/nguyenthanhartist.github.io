IMAGE PREPARATION TIPS:
-----------------------
1. Use high-quality JPG format (85-90% quality)
2. Ensure proper color profile (sRGB)
3. Crop images professionally
4. Maintain consistent aspect ratios within each series
5. Optimize file sizes (compress without quality loss)
6. Keep hero images under 500KB if possible
7. Keep artwork images under 300KB if possible

NAMING CONVENTION:

- Use hyphens (-) instead of spaces or underscores
- Use .jpg extension (lowercase)
